# OpenTelemetry in Kubernetes Platforms

Broadly, OpenTelemetry integrates into the Kubernetes ecosystem in two ways:

1. Through tooling to **monitor and profile applications** running on Kubernetes clusters, and 
2. Through **telemetry data about what Kubernetes components themselves are doing**.

Often, cloud native applications designed for Kubernetes will interact with the Kubernetes API, which makes both types of data extremely useful in investigating performance problems, deployment issues, scaling difficulties, or other production incidents.
